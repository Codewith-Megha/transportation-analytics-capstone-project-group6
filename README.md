# Transportation Analytics Pipeline

## Project Overview

This project implements an end-to-end transportation analytics pipeline using NYC Taxi Trip data.

## Features

- Data ingestion
- Data cleaning
- Feature engineering
- PostgreSQL ETL pipeline
- SQL analytics
- Dashboard (coming soon)
- Machine learning (coming soon)

## Technologies

- Python
- Pandas
- PostgreSQL
- SQLAlchemy
- Psycopg2
- Plotly
- Streamlit

## Dataset

NYC TLC Trip Record Data

Download from:

https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page

Datasets are excluded from GitHub because of their size.
